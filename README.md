# Student Grade Calculator

A Java-based application that calculates a student's total marks, average percentage, and grade.

## Features
- Accepts marks for multiple subjects
- Validates marks between 0 and 100
- Calculates total marks
- Calculates average percentage
- Generates grades automatically

## Grade Criteria
- A : 90% and above
- B : 80% - 89%
- C : 70% - 79%
- D : 60% - 69%
- F : Below 60%

## Technologies Used
- Java
- Scanner Class
- Loops
- Conditional Statements
